🎬 Netflix UI Clone

A visually appealing Netflix homepage UI clone built using HTML and CSS.
This project focuses on recreating the layout and design of Netflix with responsive styling and clean structure.

🔗 Live Demo: https://netfli-clone-renuka.netlify.app/

🚀 Features
🎨 Clean and modern Netflix-inspired UI
🔥 Hover effects for interactive feel
📱 Responsive design (works on different screen sizes)

🛠️ Tech Stack
🌐 HTML5
🎨 CSS3 

To run this project locally:

# Clone the repository
git clone https://github.com/bhavsarrenuka26/NetFlixUI

# Navigate to the project folder
cd NetFlixUI

 Open index.html in browser
